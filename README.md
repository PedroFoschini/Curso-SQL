# SQL para Análise de Dados

Este repositório reúne os materiais, exercícios e projetos desenvolvidos durante meus estudos de SQL voltados para análise de dados.

O objetivo deste projeto é consolidar conhecimentos em consultas SQL, manipulação de dados e extração de insights a partir de bases relacionais, aplicando conceitos utilizados em contextos reais de análise de dados e business intelligence.

## Conteúdos abordados

- Consultas básicas (`SELECT`, `WHERE`, `ORDER BY`)
- Filtragem e agregações
- `GROUP BY` e funções agregadas
- `JOINs`
- Subqueries
- CTEs (`Common Table Expressions`)
- Funções de janela (`Window Functions`)
- Manipulação e transformação de dados
- Boas práticas em SQL para análise de dados

## Objetivo

Desenvolver proficiência em SQL como ferramenta para:

- Exploração de dados
- Análise descritiva
- Construção de consultas analíticas
- Preparação de dados para dashboards e relatórios
- Apoio à tomada de decisão baseada em dados
